treegarden_2017plots.csv was written by Cat using treegarden_2017add.csv (which is the addtogarden tab from treegarden_2017.xlsx).

It says what plots to put each plant in!

treegarden_2017plots.xlsx was written by Cat, includes column indicating if a tag needs to be made or not. The “tags” sheet indicates which individuals need tags and where to find them.

plotmaps_2017.xlsx was made by Cat. Indicates where each plant is and where to plant new plants. Green boxes are plants already in the common garden, orange boxes are plants found in the raised beds, and blue boxes are plants that are in the greenhouse.
